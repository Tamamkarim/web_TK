# (porfolio_website-Tk)

## Projektin kuvaus
Tämä projekti on henkilökohtainen portfolio-sivusto, joka on kehitetty Next.js:llä ja Tailwind CSS:llä. Sivuston tarkoituksena on esitellä ansioluettelo, tekniset taidot, aiemmat projektit ja yhteydenottotavat ammattimaisesti ja houkuttelevasti.

## Ominaisuudet
- Moderni ja responsiivinen design kaikille laitteille
- Esittelysivu
- "Tietoa minusta" -osio
- Teknisten taitojen esittely
- Saavutusten näyttäminen
- Projektigalleria, jossa projektien yksityiskohdat
- Sähköpostiyhteydenottolomake
- Tuki vaalealle ja tummalle teemalle

## Käytetyt teknologiat
- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Sivuston käynnistäminen paikallisesti
1. Varmista, että sinulla on Node.js asennettuna.
2. Asenna tarvittavat paketit:
	```bash
	npm install
	```
3. Käynnistä sivusto paikallisesti:
	```bash
	npm run dev
	```
4. Avaa selain osoitteessa:
	[http://localhost:3000](http://localhost:3000)

## Kansiot ja rakenne
- `src/app/` : Pääsivut ja yleiset tyylitiedostot
- `src/components/` : Kaikki uudelleenkäytettävät komponentit
- `src/contexts/` : Tilanhallinta (esim. teema)
- `public/images/` : Projektien ja yleiset kuvat

## Ota yhteyttä
Kaikissa kysymyksissä tai ehdotuksissa voit ottaa yhteyttä sivuston yhteydenottolomakkeen kautta tai sähköpostilla, joka löytyy sivustolta.
